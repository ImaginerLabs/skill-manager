# 代码审查助手

## 用途

帮助开发者进行系统化的代码审查，覆盖以下维度：

- **代码质量**: 命名规范、函数长度、复杂度
- **安全性**: SQL 注入、XSS、路径遍历等常见漏洞
- **性能**: 不必要的循环、内存泄漏、N+1 查询
- **最佳实践**: 设计模式、SOLID 原则、DRY

## 使用方式

提供代码片段或文件路径，我将逐项检查并给出改进建议。
