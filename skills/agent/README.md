# Agent Skills

> **Agent Skills 目录** - 个人开发者的虚拟团队

为个人开发者打造的智能 Agent 团队，每个 Agent 都是特定领域的专家，能够像真实团队成员一样自主执行任务。

---

## 🎯 Agent 团队概览

```
👥 虚拟开发团队
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   ├── Alex (代码工程师)          - 全栈代码生成、重构

   ├── Endpoint (API 设计师)        - RESTful 设计、文档生成
   └── Fuzz (边界用例大师)          - 边界测试、异常测试

   ├── Fortress (安全审计师)        - 安全漏洞、依赖检查
   └── Metric (质量分析师)          - 代码质量、技术债

🛠️ 运维支持

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
总计：11 个 Agent Skills
```

## 🚀 快速开始

### 选择 Agent

根据你的需求选择合适的 Agent：

| 需求 | 推荐 Agent | 触发方式 |
|------|-----------|---------|
| 需要写代码 | Alex | `帮我实现...` |
| 修复 Bug | Sherlock | `帮我看看这个错误` |
| 设计 API | Endpoint | `帮我设计 API` |
| 写测试 | Fuzz | `帮我生成测试用例` |
| 系统架构 | Blueprint | `帮我设计系统架构` |
| 数据库设计 | Schema | `帮我设计数据库` |
| 云架构 | Cloudy | `帮我设计云架构` |
| 性能优化 | Turbo | `帮我优化性能` |
| 安全检查 | Fortress | `帮我检查安全漏洞` |
| 质量评估 | Metric | `评估代码质量` |
| 部署配置 | Deployer | `帮我配置 CI/CD` |


## 📋 Agent 详情

### 💻 开发角色

#### Alex - 代码工程师

**优先级**：P0  
**定位**：全栈代码生成专家  
**核心能力**：需求代码化、API 实现、组件生成、代码重构  
**文件**：[code-engineer/SKILL.md](./code-engineer/SKILL.md)

#### Sherlock - 调试专家

**优先级**：P0  
**定位**：智能问题定位和修复助手  
**核心能力**：错误分析、日志诊断、修复建议、类似问题搜索  
**文件**：[debug-expert/SKILL.md](./debug-expert/SKILL.md)

#### Endpoint - API 设计师

**优先级**：P1  
**定位**：RESTful/GraphQL 接口设计专家  
**核心能力**：接口设计、文档生成、代码生成、Mock 数据  
**文件**：[api-designer/SKILL.md](./api-designer/SKILL.md)

#### Fuzz - 边界用例大师

**优先级**：P1  
**定位**：测试用例生成专家  
**核心能力**：边界生成、E2E 场景、异常测试、等价类划分  
**文件**：[edge-case-master/SKILL.md](./edge-case-master/SKILL.md)

---

### 🏗️ 架构角色

#### Blueprint - 系统架构师

**优先级**：P1  
**定位**：系统架构设计专家  
**核心能力**：架构设计、技术选型、微服务拆分、架构评审  
**文件**：[system-architect/SKILL.md](./system-architect/SKILL.md)

#### Schema - 数据架构师

**优先级**：P1  
**定位**：数据模型和存储设计专家  
**核心能力**：数据建模、查询优化、数据迁移、数据治理  
**文件**：[data-architect/SKILL.md](./data-architect/SKILL.md)

#### Cloudy - 云架构师

**优先级**：P2  
**定位**：云服务架构设计专家  
**核心能力**：云服务选型、部署架构、成本优化、高可用设计  
**文件**：[cloud-architect/SKILL.md](./cloud-architect/SKILL.md)

### ⚡ 性能与质量

#### Turbo - 性能优化师

**优先级**：P1  
**定位**：性能分析和优化专家  
**核心能力**：性能分析、前端优化、数据库优化、API 优化  
**文件**：[performance-optimizer/SKILL.md](./performance-optimizer/SKILL.md)

#### Fortress - 安全审计师

**优先级**：P0  
**定位**：安全专家和漏洞检测专家  
**核心能力**：漏洞扫描、依赖检查、安全配置、安全加固  
**文件**：[security-auditor/SKILL.md](./security-auditor/SKILL.md)

#### Metric - 质量分析师

**优先级**：P1  
**定位**：代码质量评估专家  
**核心能力**：质量评估、技术债追踪、测试覆盖、复杂度分析  
**文件**：[quality-analyst/SKILL.md](./quality-analyst/SKILL.md)

### 🛠️ 运维支持

**优先级**：P2

**核心能力**：CI/CD 配置、Docker 配置、K8s 配置、环境管理

```

   ↓

   │      ↓
   │      ├─→ Schema (数据层设计)
   │      ├─→ Cloudy (云架构设计)
   │      └─→ Endpoint (API 设计)
   │             ↓
   │             └─→ Alex (代码实现)

   └─→ Alex (直接实现)

          ├─→ Fuzz (测试生成)

                 └─→ Deployer (部署)
```
